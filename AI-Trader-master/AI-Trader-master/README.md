------------------------------------------------------------------

---- Setting Up Locally:

`pipenv sync`

`pipenv shell`

`python -m ipykernel install --user --name=aitrader`

`jupyter notebook --ip=127.0.0.1`

---- At this point you should be able to switch to your Pipenv's Python as kernel in notebook.
